About Todo List CRUD API

What is this project?

This is a demonstration project that implements a complete REST API for task management (Todo List) using Go and MongoDB. The project demonstrates CRUD operations (Create, Read, Update, Delete) in a practical way.

Key Features:

  • Create Tasks: Add new tasks with name, description, priority, and due date
  • List Tasks: View all your tasks with filters by status and priority
  • Update Tasks: Edit task information and mark as completed
  • Delete Tasks: Remove tasks that are no longer needed
  • Advanced Filters: Filter tasks by status (pending/completed) and priority (low/medium/high)
  • Statistics: View total tasks, pending, completed, and high priority count

Technologies Used:

  • Backend: Go (Golang) with Gin Web Framework
  • Database: MongoDB
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • API: RESTful API following REST standards

Data Structure:

Each task contains the following fields:

  • ID: Unique task identifier
  • Name: Task title
  • Description: Additional details about the task
  • Priority: Priority level (low, medium, high)
  • Status: Completed or pending
  • Due Date: Deadline for completion
  • Created Date: When the task was created
  • Updated Date: Last modification timestamp

API Endpoints:

  • GET /api/v1/tasks - List all tasks
  • GET /api/v1/tasks/stats - Get statistics
  • POST /api/v1/tasks - Create new task
  • PUT /api/v1/tasks/:id - Update existing task
  • PATCH /api/v1/tasks/:id/toggle - Toggle task status
  • DELETE /api/v1/tasks/:id - Delete task
